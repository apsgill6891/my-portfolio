The core rule: Values must strictly increase going right → and down ↓ — but only among values of the same type.

Classic mode — Fill all 16 cells with numbers 1–16. Every row and every column must be strictly increasing.

Dual Track mode — The grid holds 8 numbers (1–8) and 8 letters (A–H).
Numbers only compare with other numbers. Letters only compare with other letters.
Part of the puzzle is figuring out which type belongs in each blank cell — that's the new challenge layer.

Purple button = only one valid option exists (forced move).
Nudge = highlights the blank cell with the fewest options to focus your thinking.

Keyboard: Arrow keys navigate · Numbers/letters to place · Backspace erases · H = nudge · N = new game
