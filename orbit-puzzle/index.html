Orbit

A calm little logic game inspired by daily grid puzzles, but with softer rules and easier wins. Fill the board with stars until every clue settles.

  • 1Each row and column must contain exactly the number of stars shown on the edge.
  • 2Each colored zone has a tiny badge. Match that number of stars inside the zone.
  • 3Stars cannot touch orthogonally. Diagonal neighbors are allowed, so the puzzle stays easy and playful.
