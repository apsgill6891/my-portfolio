🧵 How to Play
  • Each number on the grid is a clue for one patch.
  • Draw a rectangle that contains exactly that many squares — and the clue must be inside it.
  • Drag from one corner to another to place a patch.
  • Every square must be covered — no gaps, no overlaps.
  • Tap any filled patch to remove it.
  • Use 💡 Hint to reveal one correct patch.
